# plantlover

라즈베리파이 5와 ESP32를 활용한 저비용 생활형 스마트팜 프로젝트
프로젝트 개요
여행이나 출장 등으로 장기간 집을 비우는 동안 식물을 자동으로 관리할 수 있는 IoT 기반 스마트팜 시스템입니다. 단순한 환경 모니터링을 넘어 NDVI 적외선 분석을 통한 식물 건강도 측정 기능까지 제공합니다.
프로젝트 목적
최근 '식집사(집에서 식물을 가족처럼 돌보는 사람)'라는 신조어가 등장할 만큼 1인 가구 증가와 함께 반려식물 인구가 급증하고 있습니다. 그러나 기존 스마트팜 시스템은 대규모 농가 중심으로 설계되어 일반 가정에서 도입하기 어려운 한계가 있습니다.
본 프로젝트는 다음을 목표로 합니다.

일반 가정에서 손쉽게 설치 가능한 저비용 IoT 스마트팜 시스템 구축
과학적 데이터 기반의 식물 관리 솔루션 제공
지역 커뮤니티 및 중소 농가로 확장 가능한 기술 모델 개발

주요 기능
1. 환경 모니터링
온도, 습도, 토양 수분 데이터를 실시간으로 측정하며, MQTT 프로토콜을 통해 ESP32에서 라즈베리파이로 데이터를 전송합니다. 수집된 데이터는 라즈베리파이 서버에 저장되어 장기적인 식물 생육 패턴 분석의 기반이 됩니다.
2. 자동 제어 시스템
토양 수분 임계값을 기반으로 자동 급수 펌프를 제어하며, 조도 센서와 연동하여 조명을 자동으로 ON/OFF합니다. 웹 인터페이스를 통해 원격에서 수동으로 제어하는 것도 가능합니다.
3. 데이터 시각화 및 실시간 모니터링
웹 기반 대시보드를 통해 센서 데이터를 실시간 그래프로 시각화하고, 온도, 습도, 토양 수분, 조명 상태를 통합 모니터링합니다. 누적 데이터 히스토리 및 상태 변화 이력을 제공하여 식물 관리의 효율성을 높입니다.
4. 알람 및 알림 기능
고온, 저온, 과습, 과건조 등 이상 상태가 발생하면 자동으로 감지하여 웹 또는 모바일로 알림을 전송합니다. 시스템 오류도 감지하여 즉시 대응할 수 있습니다.
5. NDVI 기반 식물 건강도 분석
NoIR 카메라 모듈을 통해 근적외선 및 가시광선 데이터를 수집하고, 정규화 식생 지수(NDVI)를 계산하여 엽록소 활성도를 분석합니다. 식물의 건강 상태를 직관적인 색상지도로 시각화하여 객관적인 평가를 가능하게 합니다.
NDVI 계산 공식은 다음과 같습니다.
NDVI = (근적외선 - 가시광선) / (근적외선 + 가시광선)
시스템 구성
하드웨어

메인 컨트롤러: 라즈베리파이 5
센서 노드: ESP32
센서: 온습도 센서, 토양 수분 센서, 조도 센서
카메라: NoIR 카메라 모듈 (근적외선 촬영)
액추에이터: 급수 펌프, LED 조명

소프트웨어

통신 프로토콜: MQTT
데이터 저장: 라즈베리파이 로컬 DB
웹 인터페이스: 실시간 대시보드
이미지 분석: NDVI 알고리즘

시스템 아키텍처
[ESP32 + 센서] --MQTT--> [라즈베리파이 5]
                              |
                              ├── 데이터 저장
                              ├── 웹 대시보드
                              └── 알림 시스템

[NoIR 카메라] ---> [라즈베리파이 5] ---> [NDVI 분석]
기대효과
개인 사용자
장기 부재 시에도 실시간 모니터링과 자동 급수 기능으로 식물을 안정적으로 관리할 수 있습니다. 라즈베리파이와 ESP32 기반의 저비용 구축이 가능하여 일반 가정에서도 경제적 부담 없이 IoT 기술을 활용할 수 있으며, NDVI 적외선 분석을 통해 육안으로 확인하기 어려운 식물의 건강 상태를 과학적으로 정량화할 수 있습니다.
기술적 확장성
수집된 환경 데이터와 NDVI 데이터는 향후 AI 학습 및 최적 생육 환경 예측 모델 개발의 기초 자료로 활용 가능합니다. NDVI 카메라의 드론 탑재를 통해 개인용 스마트팜에서 지역 농가의 대규모 작물 모니터링으로 확장할 수 있는 구조를 제시할 수 있습니다.
지역사회 기여
실습 기반의 IoT 기술 역량 강화와 함께 누구나 손쉽게 운영할 수 있는 생활형 스마트팜 생태계를 구현하여 지역사회에 대한 실질적 기여와 기술 확산의 가능성을 제시합니다. 지역 내 중소 농가나 커뮤니티 정원 관리에 적용 가능한 지산학 협력형 기술 시제품으로 발전할 수 있습니다.
